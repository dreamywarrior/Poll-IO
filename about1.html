Poll io

Welcome to poll io. This web application is designed to help you out with creating, sharing polls and also to manage the poll results. With an aim to create a user-friendly one, this app is developed to meet the users needs. You can create a private poll which could be voted only by the people with the access to it. Also you can create public polls which are open to all the people for voting. You can vote for other public / private polls (code needed) as well. At last you can manage the polls which you have created.
